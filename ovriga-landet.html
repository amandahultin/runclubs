Vanliga frågor

Frågor och svar

Run clubs utanför Stockholm och Göteborg är gratis att delta i. Du anmäler dig enkelt via Strava eller Instagram och dyker upp på startplatsen.

Run clubs i övriga landet tar emot alla — nybörjare som vill komma igång och erfarna löpare som vill ha sällskap.

Mötesplatsen varierar per stad och club — i Malmö samlas många kring Stortorget och Pildammsparken. Se varje clubs sida för exakt startplats.

De flesta run clubs kör återkommande veckopass. Frekvensen varierar — en till tre gånger i veckan är vanligt.

Utanför Stockholm och Göteborg växer löparscenen snabbt — Malmö är ett bra exempel. Längs Ribersborgsstranden, runt Pildammsparken och genom Möllevångens gator springer en växande rörelse av löpare som har hittat varandra i run clubs. Löpargrupper här är fortfarande ett relativt nytt fenomen jämfört med Stockholm och Göteborg, men scenen är tät, engagerad och växer snabbt.

MRC Malmö är stadens mest välkända run club, med återkommande pass för alla nivåer och en community som välkomnar både nybörjare och mer erfarna löpare. Sweden Runners Malmö erbjuder ledarledda löparpass med godkänt friskvårdsbidrag — ett bra alternativ för dig som vill ha struktur och som kan använda jobbet för att täcka avgiften. Gemensamt för båda är en inkluderande atmosfär och ett genuint intresse för att få fler malmöbor att springa.

Att delta i en run club här kostar i de flesta fall ingenting. Du anmäler dig via Strava eller Instagram och dyker upp på startplatsen. Scenen är liten nog för att du snabbt känner igen ansiktena — och stor nog för att det alltid händer något. Kolla in clubs nedan och hitta ditt crew.
