Fem anledningar att springa med en run club inför höstens lopp

Ska du springa Stockholm Halvmarathon eller Midnattsloppet i höst? Här är fem anledningar att testa en run club — plus sex klubbar att börja med i Stockholm och Göteborg.

Juli 2026 Av Amanda Hultin

Ska du springa din första Stockholm Halvmarathon i höst? Eller försöka slå personbästa på Midnattsloppet? Oavsett vilket lopp du tränar inför kan träningen ibland gå trögt och kännas tråkig. Här kommer några tips på hur du kan hålla motivationen uppe genom att testa en Run Club!

🗓️Du får en plan, inte bara ett mål

Många Run Clubs erbjuder specifika runs och planer inför loppen. Ju närmare race day du kommer desto mer brukar de olika gruppernas fokus ligga på springa långpass och intervaller tillsammans. Allt för att du ska vara i toppform när du står på startlinjen. Kolla in vår eventkalender för att hitta grupper och runs som passar just dig och ditt träningsupplägg.

⏰Fasta tider gör att du dyker upp

Sommaren är kanske inte en tid när du vill ha en fullspäckad kalender. Men att ha en fast tid i kalendern för det där passet på två mil eller tio tuffa trösklar kan hjälpa dig att orka dyka upp. Många av Stockholms och Göteborgs större Run Clubs behåller sina fasta tider även över sommaren.

🔥Umgås med de som bryr sig lika mycket som du

Kanske har du redan lyckats trötta ut både din familj och dina vänner genom att bara prata löpning på semestern. Spring med en run club och prata av dig. Äntligen får du hänga med människor som faktiskt bryr sig om dina nyinköpta kolfiberskor och din gedigna vätskeplan.

👋Du får löparkompisar på köpet

Löpning behöver inte vara ensamt, och de flesta run clubs handlar minst lika mycket om gemenskapen efteråt som om själva passet. Många Run Clubs kör frukost, fika eller öl efter passet. Det är lätt att hitta saker att prata om när alla redan från början delar ett stort intresse. De som leder en Run Club är många gånger superduktiga på att få dig att känna dig välkommen och som en del av gemenskapen redan från start.

🎓Du lär dig av löpare som kommit längre

Många klubbar leds eller har startats av erfarna löpare, tränare eller coacher, vilket ger tillgång till kunskap om tempo, teknik och återhämtning som annars kan ta år att lära sig på egen hand. Det finns nästan alltid någon som gått igenom samma sak som du. Därför är det lätt att få hjälp med tips på allt ifrån hur du blir uthålligare till vad du ska äta dagarna innan en tävling.

Några klubbar att börja med

Vill du testa direkt? Här är sex klubbar som funkar bra som en första runda inför hösten.

Stockholm

  • Dopest Runners Tre pass i veckan med bag drop, Liljeholmen
  • Stockholm Run Club Gratis, internationell, backad av Adidas och Red Bull, Östermalm
  • Mellqvist Run Club Nystartad, kretsar kring frukost och bra kaffe

Göteborg

  • Slowrunners Göteborg Lugnt tempo, 90 minuter i Slottsskogen
  • Göteborg Running Club Grundad 2011, intervaller i Slottsskogen och öl på puben efteråt
  • Core Run Club Socialt först, fika efteråt

Fler alternativ hittar du i Stockholm, Göteborg och Malmö. De flesta grupper är gratis och kräver ingen anmälan — enklaste sättet att testa är att bara dyka upp en gång och se hur det känns.
