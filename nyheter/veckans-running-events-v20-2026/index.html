01
Göteborg Tisdag Trail

GRC — Trail i Änggården

Trailintervaller i Botaniska, två tempogrupper och avslappnat häng på Station Linné efteråt.

Göteborg Running Club bryter av från asfalten och tar tisdagspasset till stigarna i Botaniska. Det blir två tempogrupper och en dynamisk mix av trailintervaller, ett bra sätt att bygga både fart och uthållighet inför sommarens lopp. Trailskor rekommenderas, och efter passet är det dags för ett lugnt post-GRC-häng på Station Linné.

East Run Club — 800 m med Markus

Markus från @mymarathons leder kvällens intervaller, perfekt taper inför Göteborgsvarvet.

03
Stockholm Onsdag Collab

Run Well in Stockholm × Femme Open

Social run med Vitamin Well, två distanser och priser från Lululemon via Femme Open.

Vitamin Well drar igång onsdagskvällen med en social run i kollab med Femme Open. Det finns två rundor att välja på, 6,5 km i 6:30 min/km eller 8 km i 5:40 min/km. Mateo Florez leder vägen från Sturegatan 11.

Efter passet hänger gruppen kvar med kall Vitamin Well och Barebells, och Femme Open lottar ut priser från Lululemon. Bagdrop kommer att finnas men inte plats för ombyte.

04
Stockholm Torsdag

Stadium Run Club Stockholm

Klassiskt torsdagspass från butiken på Kungsgatan med löpskolning och intervaller.

Stadium Run Club kör veckans pass från butiken på Kungsgatan 8. Upplägget är gjort så att alla kan vara med på sina egna villkor, oavsett tidigare löpvana. Det blir uppvärmning, intervaller och en lugn nedjogg.

Anmälan görs via deras Strava innan passet drar igång klockan 17:30.

ASICS FrontRunner Social Run

Lördagsrunda runt Årstaviken med ASICS FrontRunners, fika efteråt.

ASICS FrontRunners springer månadens social run från Tantolunden. Cirka 7 km runt Årstaviken, alla tempon välkomna och en fika för den som vill stanna kvar efteråt.

ON Run Club — Sneak Peek 5 km

Förstaglimt av ONs nya butik på Biblioteksgatan, dagen innan officiell öppning.

Innan ON officiellt öppnar sin nya butik på Biblioteksgatan 18 den 14 maj bjuder de in till en social 5K. Det blir två tempogrupper (5:00 och 6:00 min/km), möjlighet att testa Cloudmonster 3 Hyper och en första titt på den nya butiken. Efter rundan väntar refreshments och tid att hänga med teamet.

Onsdag 13 maj 17:15 — skotest 17:30 — start Biblioteksgatan 18
