Run Clubs för dig som tycker om att springa tidigt

Sju run clubs (och ett bageri) för dig som helst har passet avklarat innan frukost.

Juli 2026 Av Amanda Hultin

Tycker du om att gå upp tidigt och ha träningen avklarad innan frukost? Då kanske den nya trenden med run clubs som kombinerar sina pass med frukosthäng är något för dig.

Det senaste året har allt fler run clubs och caféer förstått att det där med löpning och frukost är en perfekt kombination. Redan innan dagens måsten satt igång så är du färdigtränad och har dessutom fått äta en riktigt god frukost med mysigt sällskap.

☕Varför frukost + löpning = sant

Enligt Stravas globala sammanställning för 2025 växte antalet aktiva löpargrupper med 3,5 gånger, och bland yngre löpare är gemenskapen den vanligaste anledningen att gå med, snarare än själva prestationen. Många klubbar har svarat med att göra kaffet, fikat eller en riktigt god frukost till en fast punkt efter passet, och kaféer på flera håll har hakat på trenden och börjat samarbeta med lokala run clubs.

På runclubs.se märker vi samma sak, fler run clubs runt om i landet kör tidiga morgnar med fokus på frukost. För dig som letar efter en morgonpigg run club har vi därför skapat ett filter där du kan sortera på just "frukost". Nedan hittar du några run clubs som är renodlade frukostklubbar eller kör tidiga pass då och då.

Run clubs med frukost

  • Mellqvist Run Club Superhet Run Club av Brooks. Limiterat antal deltagare och brukar bli full snabbt som bara den, så anmäl dig i tid.
  • Saucony & Downtown Camper Run Club Onsdagar 06:45 vid Downtown Camper, Brunkebergstorg. Gratis morgonpass som numera alltid rundas av med frukost.
  • ESS Runners Club Torsdagsmorgnar med start vid något av ESS hotell (tiden brukar variera lite). Sju till tio kilometer, sedan frukostmingel och prat.
  • Running For Breakfast Söndagar 09:00 i Västerås, Eskilstuna och Norrköping. Namnet säger allt, frukost till rabatterat pris väntar efteråt.
  • Fun Run Malmö Måndagar 06:00. Efter passet blir det ofta frukost eller en kaffe ihop, beroende på gänget och dagen.
  • East Run Club En av Göteborgs populäraste klubbar. Kör ofta tidiga pass som avslutas med frukost.
  • Strawberry Run Club Måndagsmorgnar på hotell i Sverige och Norge, ett par gånger i månaden. De 30 första vid start bjuds på en minifrukost.

Bonus: ingen frukost, men med ändå

  • Svedjans Löpsällskap Egentligen ingen frukostklubb. Måndagar 17:30 vid bageriet på Södermalm, och ibland blir det Bun Run, löpning och bullar i samma pass.

Fler löparklubbar hittar du i Stockholm, Göteborg och Övriga landet. De flesta är gratis och kräver ingen anmälan, det enklaste sättet att testa är att bara dyka upp.
